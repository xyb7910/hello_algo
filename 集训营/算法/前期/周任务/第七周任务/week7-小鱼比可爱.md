# week7-小鱼比可爱

## 题目描述

人比人，气死人；鱼比鱼，难死鱼。小鱼最近参加了一个“比可爱”比赛，比的是每只鱼的可爱程度。

参赛的鱼被从左到右排成一排，有的小鱼头朝左边，有的小鱼头朝右边，然后每只鱼会得到一个整数数值，表示这只鱼的可爱程度，很显然整数越大，表示这只鱼越可爱，而且任意两只鱼的可爱程度**可能一样**。

鱼头朝向左边的鱼，只鱼只能看见在它左边的鱼的可爱程度；鱼头朝向右边的鱼，只鱼只能看见在它右边的鱼的可爱程度。它们心里都在计算，在自己的眼力范围内有多少只鱼**不如自己可爱**。

请你帮这些可爱但是鱼脑不够用的小鱼们计算一下。

## 输入格式

第一行输入一个正整数 $n$，表示鱼的数目。

第二行内输入 $n$ 个正整数，用空格间隔，依次表示从左到右每只小鱼的可爱程度 $a_i$。

第三行内输入 $n$ 个**字符** $c_i$，用空格间隔，依次表示每条小鱼头的朝向。若 $c_i=\text{L}$，则表示第 $i$ 条小鱼头朝左边; 若 $c_i=\text{R}$，则表示第 $i$ 条小鱼头朝右边。

## 输出格式

一行，输出 $n$ 个整数，用空格间隔，依次表示每只小鱼眼中有多少只鱼不如自己可爱。

## 样例 #1

### 样例输入 #1

```
6
4 3 0 5 1 2
L L L L L L
```

### 样例输出 #1

```
0 0 0 3 1 2
```

## 样例 #2

### 样例输入 #2

```
5
6 2 3 4 5
R R R L R
```

### 样例输出 #2

```
4 0 0 2 0
```

## 提示

对于 $20\%$ 的数据，满足 $c_i=\text{L}$；

对于 另外$20\%$ 的数据，满足 $c_i=\text{R}$；

对于 $100\%$ 的数据，满足 $n\le 5000$，$0\le a_i\le 10^9$，$c_i=\text{L或R}$
