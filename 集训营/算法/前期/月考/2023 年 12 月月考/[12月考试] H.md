# [12月考试] H

## 题目描述

Z 的栅栏褪色了，Z 现在要重新刷栅栏。

Z 的栅栏目前形如字符串 $s$，$s$ 由小写英文字母和 `*` 组成。`*` 表示褪色的栅栏，小写英文字母代表了一片有色的栅栏，其颜色用该字母表示。

Z 现在要给所有的褪色的栅栏刷上一种颜色（他只能刷褪色的栅栏，而不能刷有色的栅栏）。Z 有两把刷子，第一把刷子可以蘸 $26$ 种颜料（分别用 $26$ 个小写字母代表），每次只能刷一个褪色栅栏，第二把刷子只能蘸彩虹色（彩虹色是一种颜色，这种颜色不同于任何一种字母代表的颜色），会给相邻的两个褪色栅栏分别刷上彩虹色。求 Z 将所有的褪色的栅栏刷上颜色的方案数，对 $998244353$ 取模。

## 输入格式

一行一个字符串 $s$，表示 Z 的栅栏的目前的状态。

## 输出格式

一行一个整数，表示求得的方案数对 $998244353$ 取模的结果。

## 样例 #1

### 样例输入 #1

```
r*ed**bag*
```

### 样例输出 #1

```
457652
```

## 提示

### 样例解释 

若 $4$ 个褪色栅栏均使用第一把刷子，有 $26\times 26\times 26\times 26=456976$ 种方案。  
若第 $2,3$ 个褪色栅栏使用第二把刷子，则额外有 $26\times 26=676$ 种方案。  

共计 $456976 + 676 = 457652$ 种。

### 数据规模与约定

对于前 $10\%$ 的数据，满足字符串 $s$ 中不包括 `*`；  
对于另外 $10\%$ 的数据，满足字符串 $s$ 中只包括一个 `*`；  
对于另外 $20\%$ 的数据，满足字符串 $s$ 中只包括两个 `*`；  
对于另外 $30\%$ 的数据，满足字符串 $s$ 中 `*` 不超过 $15$ 个；  
对于 $100\%$ 的数据，满足字符串 $s$ 的长度不超过 $5\times 10^5$，字符串 $s$ 中只包括小写英文字母和 `*`。
