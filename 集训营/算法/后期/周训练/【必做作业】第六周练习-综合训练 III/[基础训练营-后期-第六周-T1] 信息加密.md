# [基础训练营-后期-第六周-T1] 信息加密

## 题目背景

> 数据速朽，唯安全永恒。

## 题目描述

信息安全永远是重中之重。在本题中，你需要实现一个简单的加密算法。

给出一个由 $0,1$ 组成的字符串 $S$，请将其中所有的 $0$ 换成 $1$，所有的 $1$ 换成 $0$，并输出新的字符串 $S'$。

## 输入格式

输入一行一个字符串 $S$。

## 输出格式

输出一行一个字符串 $S'$。

## 样例 #1

### 样例输入 #1

```
0011
```

### 样例输出 #1

```
1100
```

## 提示

- 对于 $30\%$ 的数据，$|S| \le 10^3$；
 - 对于 $100\%$ 的数据，$|S| \le 10^6$。

其中 $|S|$ 代表字符串 $S$ 的长度。
