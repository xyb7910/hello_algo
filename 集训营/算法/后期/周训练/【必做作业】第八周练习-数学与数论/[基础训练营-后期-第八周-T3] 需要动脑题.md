# [基础训练营-后期-第八周-T3] 需要动脑题

## 题目描述

给出任意一组 $x,y$，使得 $x^y \cdot y + y^x \cdot x=N$。

## 输入格式

**每组测试点中包含多组测试数据。**

输入的第一行为一个整数 $T$，代表测试数据组数。

对于每组测试数据，输入一行一个整数 $N$。

## 输出格式

对于每组测试数据，输出一行两个整数 $x,y$，由空格隔开。

**你只需要给出任意一组合法解。如果没有合法解，输出两个 $-1$。**

## 样例 #1

### 样例输入 #1

```
2
42
13
```

### 样例输出 #1

```
2 3
-1 -1
```

## 提示

- 对于 $30\%$ 的测试点，$1 \le N \le 20$；
 - 对于 $100\%$ 的测试点，$1 \le N \le 10^{9}$，$1 \le T \le 100$。
