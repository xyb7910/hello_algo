# [基础训练营-后期-第七周-T3] 栈上游戏

## 题目描述

小 A 有一个栈，栈里面有 $n$ 个正整数元素 $a_1,a_2\dots a_n$，从栈顶到栈底排列。

小 A 现在能够对栈进行 $k$ 次操作。每次操作为下面两种方式二选一：

- 弹出栈顶；
- 塞入一个元素。这个元素必须是从栈中被弹出过的。在其再一次被弹出栈之前，这个元素不能再被塞入；

小 A 希望知道在这种情况下栈顶元素最大是多少？如果栈一定为空，则输出 $-1$。

## 输入格式

第一行输入两个正整数 $n$ 和 $k$，表示栈内的元素个数；

第二行输入 $n$ 个正整数 $a_1,a_2,\dots a_n$。

## 输出格式

输出一行，表示栈顶元素最大值。特别的，如果栈一定为空，则输出 $-1$。

## 样例 #1

### 样例输入 #1

```
6 2
5 2 2 4 1 6
```

### 样例输出 #1

```
5
```

## 提示

**【数据范围】**

对于 $30\%$ 的数据，$1 \leq n,k \leq 10$；

对于 $70\%$ 的数据，$1 \leq n,k \leq 1000$；

对于所有数据，$1 \leq n,k \leq 10^5$，$1 \leq a_i \leq 10^9$。
