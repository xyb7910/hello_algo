# week10-[一维数组]气候

## 题目描述

定义一天为干燥的，当且仅当当天降雨量小于 $k$。

一共给出 $n$ 天的降雨量，第 $i$ 天的降雨量为 $a_i$，求最长有几天是连续干燥的。

对于所有数据，$n\leq 10^5$，$1\leq a_i,k<2^{64}$。

## 输入格式

输入共 $2$ 行。

第 $1$ 行输入 $2$ 个正整数 $n,k$。

第 $2$ 行输入 $n$ 个正整数 $a_1,a_2,\ldots,a_n$。

## 输出格式

输出共 $1$ 行 $1$ 个整数表示答案。

## 样例 #1

### 样例输入 #1

```
7 4
6 3 4 1 2 3 4
```

### 样例输出 #1

```
3
```

## 提示

对于所有数据，$n\leq 10^5$，$1\leq a_i,k<2^{64}$。
