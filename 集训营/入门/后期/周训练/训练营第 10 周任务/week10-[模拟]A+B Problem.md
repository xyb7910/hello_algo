# week10-[模拟]A+B Problem

## 题目描述

给定 $a,b$，输出 $a+b$ 的竖式计算图，输出格式见样例。

对于所有数据，$1\leq a,b<2^{63}$。

## 输入格式

输入共 $1$ 行 $2$ 个正整数 $a,b$。

## 输出格式

输出共 $4$ 行字符串，表示竖式计算图。

## 样例 #1

### 样例输入 #1

```
198 213
```

### 样例输出 #1

```
198
+ 213
-----
  411
```

## 样例 #2

### 样例输入 #2

```
999 2
```

### 样例输出 #2

```
999
+   2
-----
 1001
```

## 提示

对于所有数据，$1\leq a,b<2^{63}$。

提示：数据类型可以使用 unsigned long long。
