# week12-[一维数组]移山

## 题目描述

有 $n$ 座山，每座山的高度为 $a_{i}$，你现在想要令这 $n$ 座山的高度相同，但是高度之和不变。

请问每座山的高度变化量的绝对值之和为多少。

## 输入格式

第一行一个正整数 $n$，表示山的数量。

第二行共 $n$ 个正整数，第 $i$ 个正整数 $a_{i}$ 表示山的高度。

## 输出格式

一行一个正整数表示问题的答案。

## 样例 #1

### 样例输入 #1

```
5
6 5 8 5 6
```

### 样例输出 #1

```
4
```

## 提示

$1\leq n\leq 100,1\leq a_{i}\leq 10000$

保证 $a_{i}$ 之和的平均值为整数。
