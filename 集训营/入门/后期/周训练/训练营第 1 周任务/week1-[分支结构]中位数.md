# week1-[分支结构]中位数

## 题目描述

给定 $4$ 个正整数 $a,b,c,d$，输出它们的中位数，答案四舍五入保留 $1$ 位小数。

## 输入格式

输入共 $1$ 行 $4$ 个正整数 $a,b,c,d$。

## 输出格式

输出共 $1$ 行 $1$ 个浮点数表示答案。

## 样例 #1

### 样例输入 #1

```
3 1 2 7
```

### 样例输出 #1

```
2.5
```

## 提示

#### 样例解释 $1$

$(2 + 3) \div 2 = 2.5$。

### 数据范围

对于所有数据，$1\leq a,b,c,d\leq 100$。
