# week5-[循环结构]听歌

## 题目描述

小 A 有 $n$ 首喜欢的歌，第 $i$ 首歌的长度为 $a_i$ 分钟。第 $i$ 首歌要听 $i$ 遍，问最后总共听了多少分钟？

## 输入格式

输入共 $2$ 行。

第 $1$ 行 $1$ 个正整数 $n$。

第 $2$ 行 $n$ 个正整数 $a_i$。

## 输出格式

输出共 $1$ 行 $1$ 个正整数表示答案。

## 样例 #1

### 样例输入 #1

```
3
3 2 1
```

### 样例输出 #1

```
10
```

## 提示

### 数据范围

对于所有数据，$1 \leq n,a_i\leq 100$。
