# week5-[一维数组]去重

## 题目描述

给定正整数 $n$ 和 $n$ 个正整数 $a_i$，请将 $a_i$ 中重复的元素只保留一个，然后保留原顺序输出。

## 输入格式

输入共 $2$ 行。

第 $1$ 行 $1$ 个正整数 $n$。

第 $2$ 行 $n$ 个正整数 $a_i$。

## 输出格式

输出共 $1$ 行若干个正整数表示答案。

## 样例 #1

### 样例输入 #1

```
7
3 4 5 4 3 1 3
```

### 样例输出 #1

```
3 4 5 1
```

## 提示

### 数据范围

对于所有数据，$1 \leq n,a_i\leq 100$。
