# week5-[字符数组]查找

## 题目描述

给定两个只包含小写字母的字符串 $a,b$，请问 $b$ 是否在 $a$ 中出现过？

## 输入格式

输入共 $2$ 行。

第 $1$ 行 $1$ 个只包含小写字母的字符串 $a$。

第 $2$ 行 $1$ 个只包含小写字母的字符串 $b$。

## 输出格式

输出共 $1$ 行表示答案。如果出现过，输出 `YE5`，否则输出 `NO`。

## 样例 #1

### 样例输入 #1

```
helloworld
owo
```

### 样例输出 #1

```
YE5
```

## 样例 #2

### 样例输入 #2

```
abcdcaa
da
```

### 样例输出 #2

```
NO
```

## 提示

### 数据范围

对于所有数据，$a,b$ 的长度都不超过 $1000$。
