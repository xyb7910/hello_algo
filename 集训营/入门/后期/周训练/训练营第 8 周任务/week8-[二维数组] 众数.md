# week8-[二维数组] 众数

## 题目描述

给一个边长为 $n$ 的正方形二维数组 $a$， 求出数组中出现最多的数出现的次数。

## 输入格式

第一行一个整数 $n$ 。

接下来 $n$ 行每行 $n$ 个元素表示数组 。

## 输出格式

一行一个整数表示答案。

## 样例 #1

### 样例输入 #1

```
4
46 45 20 7
48 25 16 20
43 18 7 14
40 39 11 34
```

### 样例输出 #1

```
2
```

## 提示

$1 \leq  n \leq 1000, 1 \leq a_{i, j} \leq 100$
