# week7-[一维数组]小A的一维数组

## 题目描述

小 A 有一个包含 $n$ 个正整数的数组 $a$，数组中的第 $i$ 个数记为 $a_i$。

给定正整数 $k$，对于数组中的每一个数 $a_i$，输出数组 $a$ 中有多少个数与 $a_i$ 的差的绝对值不大于 $k$。

对于所有数据，$n,k,a_i\leq 10^3$。

## 输入格式

输入共 $2$ 行。

第 $1$ 行输入 $2$ 个正整数 $n,k$。

第 $2$ 行输入 $n$ 个正整数 $a_i$。

## 输出格式

输出共 $1$ 行 $n$ 个正整数，第 $i$ 个数表示对于 $a_i$ 所求的答案。

## 样例 #1

### 样例输入 #1

```
5 1
2 5 3 1 1
```

### 样例输出 #1

```
4 1 2 3 3
```

## 提示

对于所有数据，$n,k,a_i\leq 10^3$。
