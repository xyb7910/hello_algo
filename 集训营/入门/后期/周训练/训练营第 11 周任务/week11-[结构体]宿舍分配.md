# week11-[结构体]宿舍分配

## 题目描述

学校新开学一共接收了 $n$ 名住校生，第 $i$ 名住校生的学号为 $a_{i}$，分配的寝室编号为 $b_{i}$

你需要按照学号从小到大依次输出这 $n$ 名住校生的学号和寝室编号。

## 输入格式

第一行一个正整数 $n$

接下来 $n$ 行，每行两个正整数 $a_{i},b_{i}$，表示一个学生的信息。

保证学生的学号两两不同。

## 输出格式

输出 $n$ 行，每行两个正整数，表示学生的学号和寝室编号。

需要注意输出的学号按输出顺序递增。

## 样例 #1

### 样例输入 #1

```
5
7 59
3 11
1 65
9 55
6 25
```

### 样例输出 #1

```
1 65
3 11
6 25
7 59
9 55
```

## 提示

$1\leq n\leq 100,1\leq a_{i}\leq 1000,1\leq b_{i}\leq 100$
