# week3-[二维数组]最大列

## 题目描述

给定一个 $n$ 行 $m$ 的二维数组 $a_{ij}$，找出其中的一列使得这列的元素和最大。输出这列的元素和。

## 输入格式

输入共 $n + 1$ 行。

第 $1$ 行 $2$ 个正整数 $n,m$。

接下来 $n$ 行，每行 $m$ 个用空格隔开的正整数 $a_{ij}$。

## 输出格式

输出共 $1$ 行 $1$ 个整数表示答案。

## 样例 #1

### 样例输入 #1

```
3 4
1 2 3 4
4 3 2 1
3 2 2 3
```

### 样例输出 #1

```
8
```

## 提示

#### 样例解释 $1$

第 $1$ 列与第 $4$ 列的元素和都是 $8$。

### 数据范围

对于所有数据，$1\leq n,m,a_{ij}\leq 100$。
