# [10月考试] A

## 题目描述

小 A 去参加考试，考试共有 $n$ 门科目，每一门科目的分数为 $2\sim 5$ 分。

已知小 A 共 $n$ 门科目的分数总和为 $m$，求小 A 至少有几门科目考了 $2$ 分。

对于所有数据，$1\leq n\leq 100$，$2n\leq m \leq 5n$。

## 输入格式

输入共 $1$ 行 $2$ 个正整数 $n,m$。

## 输出格式

输出共 $1$ 行 $1$ 个整数，表示答案。

## 样例 #1

### 样例输入 #1

```
2 5
```

### 样例输出 #1

```
1
```

## 样例 #2

### 样例输入 #2

```
3 8
```

### 样例输出 #2

```
1
```

## 样例 #3

### 样例输入 #3

```
4 12
```

### 样例输出 #3

```
0
```

## 提示

对于所有数据，$1\leq n\leq 100$，$2n\leq m \leq 5n$。
