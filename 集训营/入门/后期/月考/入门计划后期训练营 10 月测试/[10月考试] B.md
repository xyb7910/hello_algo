# [10月考试] B

## 题目描述

给定 $1$ 个三位正整数 $x$，求 $x$ 的各位数是否全都不相等。

如果全都不相等，输出 `Yes`；否则，输出 `No`。

对于所有数据，$100\leq x\leq 999$。

## 输入格式

输入共 $1$ 行 $1$ 个正整数 $x$。

## 输出格式

输出共 $1$ 行 $1$ 个字符串，表示答案。

## 样例 #1

### 样例输入 #1

```
704
```

### 样例输出 #1

```
Yes
```

## 样例 #2

### 样例输入 #2

```
477
```

### 样例输出 #2

```
No
```

## 提示

对于所有数据，$100\leq x\leq 999$。
