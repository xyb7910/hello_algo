# [9月考试] A

## 题目描述

小 A 在洛谷 AC 了 $x$ 道题，WA 了 $y$ 道题，TLE 了 $z$ 道题。

请求出小 A 所有做过的题目的 AC 率（$\text{AC 率}=\frac{\text{AC 题数}}{\text{AC+WA+TLE 题数}}$），答案四舍五入保留 $4$ 位小数。

对于所有数据，$1\leq x,y,z\leq 100$。

## 输入格式

输入共 $1$ 行 $3$ 个正整数 $x,y,z$。

## 输出格式

输出共 $1$ 行 $1$ 个浮点数，表示答案。

## 样例 #1

### 样例输入 #1

```
3 4 10
```

### 样例输出 #1

```
0.1765
```

## 提示

对于所有数据，$1\leq x,y,z\leq 100$。
