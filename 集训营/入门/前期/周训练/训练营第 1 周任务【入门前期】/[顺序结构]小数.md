# [顺序结构]小数

## 题目描述

给定一个浮点数 $x$，请将其四舍五入保留两位小数输出。

## 输入格式

输入共 $1$ 行 $1$ 个浮点数，表示原始浮点数。

## 输出格式

输出共 $1$ 行 $1$ 个浮点数，表示四舍五入保留两位小数的浮点数。

## 样例 #1

### 样例输入 #1

```
17.288
```

### 样例输出 #1

```
17.29
```

## 样例 #2

### 样例输入 #2

```
1.7
```

### 样例输出 #2

```
1.70
```

## 提示

### 数据范围

对于所有数据，$0\leq x\leq 1000$，原始小数位不超过 $6$ 位。
