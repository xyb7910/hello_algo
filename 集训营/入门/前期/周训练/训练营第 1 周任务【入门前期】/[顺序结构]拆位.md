# [顺序结构]拆位

## 题目描述

有一个正整数 $n$，并且 $n$ 在 $1000$ 到 $9999$ 之间。

请输出 $n$ 的千位数与百位数之和。

## 输入格式

输入共 $1$ 行 $1$ 个正整数，表示 $n$。

## 输出格式

输出共 $1$ 行 $1$ 个正整数，表示所求答案。

## 样例 #1

### 样例输入 #1

```
3201
```

### 样例输出 #1

```
5
```

## 样例 #2

### 样例输入 #2

```
7286
```

### 样例输出 #2

```
9
```

## 提示

#### 样例解释 1

$3201$ 的千位数是 $3$，百位数是 $2$，所以答案为 $3+2=5$。

### 数据范围

对于所有数据，$1000\leq n\leq 9999$。
