# [循环嵌套]填数

## 题目描述

找出一组非负整数 $a,b,c,d$，满足 $a+b+c+d=k$，且 $(a+2b)\times (c+3d) + (3a+d)\times (2c+b)$ 最大。输出这个最大值。

## 输入格式

输入共 $1$ 行 $1$ 个正整数 $k$。

## 输出格式

输出共 $1$ 行 $1$ 个正整数表示答案。

## 样例 #1

### 样例输入 #1

```
2
```

### 样例输出 #1

```
7
```

## 提示

#### 样例解释 $1$

$a=0,b=1,c=0,d=1$。

### 数据范围

对于所有数据，$1 \leq k\leq 50$。
