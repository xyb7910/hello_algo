# [分支结构]日期

## 题目描述

$y$ 年 $m$ 月有多少日？

年份 $y$ 其为闰年当且仅当满足以下两种情况之一：

- $y$ 能被 $4$ 整除且不能被 $100$ 整除；
- $y$ 能被 $400$ 整除。

## 输入格式

输入共 $1$ 行 $2$ 个正整数 $y,m$。

## 输出格式

输出共 $1$ 行 $1$ 个正整数表示答案。

## 样例 #1

### 样例输入 #1

```
1900 2
```

### 样例输出 #1

```
28
```

## 样例 #2

### 样例输入 #2

```
2003 5
```

### 样例输出 #2

```
31
```

## 提示

### 数据范围

对于所有数据，$1900 \leq y\leq 2300$，$1 \leq m \leq 12$。
