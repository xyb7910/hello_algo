# [分支结构]分数

## 题目描述

给定整数 $a,b$，判断 $\frac a b$ 是否存在。

## 输入格式

输入共 $1$ 行 $2$ 个正整数 $a,b$。

## 输出格式

输出共 $1$ 行表示答案。如果存在，输出 `0K`，否则输出 `NO`。

## 样例 #1

### 样例输入 #1

```
8 6
```

### 样例输出 #1

```
0K
```

## 样例 #2

### 样例输入 #2

```
4 0
```

### 样例输出 #2

```
NO
```

## 提示

### 数据范围

对于所有数据，$0 \leq a,b\leq 100$。
