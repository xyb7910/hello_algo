# [循环结构]求平均数

## 题目描述

给定 $n$ 个数，求出他们的平均数，答案四舍五入保留 $1$ 位小数。

## 输入格式

输入共 $2$ 行。

第 $1$ 行输入 $1$ 正整数 $n$。

第 $2$ 行输入 $n$ 个正整数 $a_i$。

## 输出格式

输出共 $1$ 行 $1$ 个浮点数，表示答案。

## 样例 #1

### 样例输入 #1

```
6
5 1 1 4 4 1
```

### 样例输出 #1

```
2.7
```

## 提示

#### 样例解释 $1$

$(5+1+1+4+4+1)\div 6 = \frac {8}3 \approx 2.7$

### 数据范围

对于所有数据，$n\leq 100$，$1\leq a_i\leq 100$。
