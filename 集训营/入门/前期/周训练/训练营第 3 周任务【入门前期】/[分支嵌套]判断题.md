# [分支嵌套]判断题

## 题目描述

给定一个正整数 $x$，如果 $x>100$，若 $x$ 是偶数，输出 "even>100"；若 $x$ 是奇数，输出 "odd>100"；如果 $x\leq 100$，若 $x$ 是偶数，输出 "even<=100"；若 $x$ 是奇数，输出 "odd<=100"。

## 输入格式

输入共 $1$ 行 $1$ 个正整数 $x$。

## 输出格式

输出共 $1$ 行 $1$ 个字符串。

## 样例 #1

### 样例输入 #1

```
101
```

### 样例输出 #1

```
odd>100
```

## 样例 #2

### 样例输入 #2

```
102
```

### 样例输出 #2

```
even>100
```

## 样例 #3

### 样例输入 #3

```
100
```

### 样例输出 #3

```
even<=100
```

## 样例 #4

### 样例输入 #4

```
99
```

### 样例输出 #4

```
odd<=100
```

## 提示

### 数据范围

对于所有数据，$x\leq 200$。
