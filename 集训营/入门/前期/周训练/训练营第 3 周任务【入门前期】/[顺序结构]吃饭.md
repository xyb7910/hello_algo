# [顺序结构]吃饭

## 题目描述

小 A 一顿饭要吃 $a$ 种肉，$b$ 种蔬菜。

买一种肉需要 $x$ 元，买一种蔬菜需要 $y$ 元，请问小 A 一共要花多少钱呢？

## 输入格式

输入共 $1$ 行 $4$ 个正整数，表示 $a,b,x,y$。

## 输出格式

输入共 $1$ 行 $1$ 个正整数，表示答案。

## 样例 #1

### 样例输入 #1

```
2 3 4 5
```

### 样例输出 #1

```
23
```

## 提示

#### 样例解释 1

小 A 需要花 $2\times 4=8$ 元买肉，$3\times 5=15$ 元买菜，一共需要 $8+15=23$ 元。

### 数据范围

对于所有数据，$a,b,x,y\leq 100$。
