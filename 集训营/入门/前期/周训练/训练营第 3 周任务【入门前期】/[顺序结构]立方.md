# [顺序结构]立方

## 题目描述

定义 $x$ 的立方为 $x\times x\times x$。

输入正整数 $a,b$，求出 $a$ 的立方和 $b$ 的立方。

## 输入格式

输入共 $1$ 行 $2$ 个正整数，表示 $a,b$。

## 输出格式

输出共 $1$ 行 $2$ 个正整数，表示 $a$ 的立方和 $b$ 的立方。

## 样例 #1

### 样例输入 #1

```
2 3
```

### 样例输出 #1

```
8 27
```

## 提示

#### 样例解释 1

$2\times 2\times 2=8$，$3\times 3\times 3=27$。

### 数据范围

对于所有数据，$a,b\leq 10$。
