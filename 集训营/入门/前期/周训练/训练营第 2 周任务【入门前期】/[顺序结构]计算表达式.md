# [顺序结构]计算表达式

## 题目描述

给定正整数 $a,b,c,d$，请求出 $(a+b)\times (c-d)$ 的值。

## 输入格式

输入共 $1$ 行 $4$ 个正整数，表示 $a,b,c,d$。

## 输出格式

输出共 $1$ 行 $1$ 个整数，表示答案。

## 样例 #1

### 样例输入 #1

```
2 3 7 5
```

### 样例输出 #1

```
10
```

## 样例 #2

### 样例输入 #2

```
1 2 3 4
```

### 样例输出 #2

```
-3
```

## 提示

### 数据范围

对于所有数据，$1\leq a,b,c,d\leq 100$。
