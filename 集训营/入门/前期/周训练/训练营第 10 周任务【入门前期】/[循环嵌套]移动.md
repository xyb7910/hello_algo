# [循环嵌套]移动

## 题目描述

给定 $n$ 和 $a_1,a_2,a_3,\ldots,a_n$，问有多少个 $k$ 满足 $a_k=a_{k+2}$？其中 $1\leq k\leq n-2$。

## 输入格式

输入共 $2$ 行。

第 $1$ 行 $1$ 个正整数 $n$。

第 $2$ 行 $n$ 个正整数 $a_i$。

## 输出格式

输出共 $1$ 行表示答案。

## 样例 #1

### 样例输入 #1

```
5
1 1 1 2 2
```

### 样例输出 #1

```
1
```

## 提示

### 数据范围

对于所有数据，$1 \leq n,a_i\leq 1000$。
