# [分支结构]数字组成

## 题目描述

读入 $5$ 个字符，判断它们是否按顺序组成了一个五位数。

## 输入格式

输入共 $1$ 行 $5$ 个连续的字符。

## 输出格式

输出共 $1$ 行表示答案。如果组成了五位数，输出 `fiiiiIi1IIi1li1ll1l`，否则输出 `cCcCCCcccc77Cc7A`。

## 样例 #1

### 样例输入 #1

```
15672
```

### 样例输出 #1

```
fiiiiIi1IIi1li1ll1l
```

## 样例 #2

### 样例输入 #2

```
1A3(3
```

### 样例输出 #2

```
cCcCCCcccc77Cc7A
```

## 样例 #3

### 样例输入 #3

```
01234
```

### 样例输出 #3

```
cCcCCCcccc77Cc7A
```

## 提示

#### 样例解释 $3$

$1234$ 是四位数不是五位数。

### 数据范围

对于所有数据，保证输入了 $5$ 个可读字符。
