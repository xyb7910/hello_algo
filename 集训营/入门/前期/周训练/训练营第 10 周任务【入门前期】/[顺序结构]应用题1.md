# [顺序结构]应用题1

## 题目描述

从 A 点走到 B 点，已经走了 $x$ m，是**剩下**路程的 $k$ 倍，问 AB 距离多少 m，答案四舍五入保留 $2$ 位小数。

## 输入格式

输入共 $1$ 行 $2$ 个正整数 $x,k$。

## 输出格式

输出共 $1$ 行 $1$ 个浮点数，表示答案。

## 样例 #1

### 样例输入 #1

```
10 2
```

### 样例输出 #1

```
15.00
```

## 提示

#### 样例解释 1

剩下路程为 $10\div 2=5$ (m)，一共 AB 距离 $10+5=15$ (m)。

### 数据范围

对于所有数据，$1\leq x,k\leq 100$。
