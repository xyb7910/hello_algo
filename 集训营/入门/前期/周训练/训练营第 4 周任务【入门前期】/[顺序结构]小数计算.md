# [顺序结构]小数计算

## 题目描述

给定浮点数 $a,b$，输出 $a+b$ 和 $a\times b$，答案四舍五入保留 $2$ 位小数。

## 输入格式

输入共 $1$ 行 $2$ 个浮点数，表示 $a,b$。

## 输出格式

输入共 $1$ 行 $2$ 个浮点数，表示 $a+b,a\times b$。

## 样例 #1

### 样例输入 #1

```
1.100 1.100
```

### 样例输出 #1

```
2.20 1.21
```

## 提示

### 数据范围

对于所有数据，$0\leq a,b\leq 10$，$a,b$ 小数点后给出 $3$ 位。
