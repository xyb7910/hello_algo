# [循环嵌套]铺地板

## 题目描述

给定正整数 $n$ 和 $m$，输出一个 $n \times m$ 的瓷砖地板，格式参见样例。

## 输入格式

输入共 $1$ 行 $2$ 个正整数 $n,m$。

## 输出格式

输出共 $n$ 行表示答案。

## 样例 #1

### 样例输入 #1

```
4 5
```

### 样例输出 #1

```
.*.*.
*.*.*
.*.*.
*.*.*
```

## 样例 #2

### 样例输入 #2

```
7 7
```

### 样例输出 #2

```
.*.*.*.
*.*.*.*
.*.*.*.
*.*.*.*
.*.*.*.
*.*.*.*
.*.*.*.
```

## 提示

### 数据范围

对于所有数据，$1 \leq n,m\leq 100$。
