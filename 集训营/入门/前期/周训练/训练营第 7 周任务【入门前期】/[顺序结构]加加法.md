# [顺序结构]加加法

## 题目描述

给定 $2$ 个浮点数 $a,b$，令 $x$ 为它们整数部分的和，求 'A' 之后第 $x$ 个字母。

'A' 之后第 $1$ 个是 'B'，第 $2$ 个是 'C'，以此类推。

## 输入格式

输入共 $1$ 行 $2$ 个浮点数 $a,b$。

## 输出格式

输出共 $1$ 行 $1$ 个字符，表示答案。

## 样例 #1

### 样例输入 #1

```
8.99 0.99
```

### 样例输出 #1

```
I
```

## 提示

#### 样例解释 1

$8+0=8$，'A' 后第 $8$ 个字母是 'I'。

### 数据范围

对于所有数据，$0\leq a,b\leq 10$，$a,b$ 含至多 $3$ 位小数位。
