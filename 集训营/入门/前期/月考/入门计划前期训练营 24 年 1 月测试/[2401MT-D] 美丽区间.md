# [2401MT-D] 美丽区间

## 题目描述

小 F 喜欢区间，她认为一些数构成美丽区间，当且仅当以下条件被全部满足：

- 这些数都是正整数。
- 这些数一共有 $5$ 个。
- 这些数从小到大排列，两两相差 $2$。
- 所有的数都小于 $n$。
- $5$ 个数的和，十位的值为 $3$。

例如，$2\ 4\ 6\ 8\ 10$ 是满足要求的一个美丽区间，因为它满足了上面的全部要求。

现在给出 $n$，请你求出美丽区间的数目。

## 输入格式

输入一行一个整数 $n$。

## 输出格式

输出一行一个整数，表示美丽区间的数目。

## 样例 #1

### 样例输入 #1

```
30
```

### 样例输出 #1

```
2
```

## 样例 #2

### 样例输入 #2

```
1000
```

### 样例输出 #2

```
100
```

## 提示

- 对于 $30\%$ 的测试数据，$n \le 1000$。
- 对于 $100\%$ 的测试数据，$n \le 10^7$。
