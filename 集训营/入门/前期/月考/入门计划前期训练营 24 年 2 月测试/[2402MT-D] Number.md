# [2402MT-D] Number

## 题目描述

定义一个数 $x$ 是好数当且仅当 $x$ 各数位上的数按照高位到低位递增，例如 $12356$ 是一个好数，而 $1233,1253$ 不是好数。

小 E 想知道给定 $l,r$，$[l,r]$ 内有多少个好数。

对于所有数据，$1\leq l\leq r\leq 10^6$。

## 输入格式

输入共 $1$ 行 $2$ 个正整数 $l,r$。

## 输出格式

输出共 $1$ 行 $1$ 个整数，表示答案。

## 样例 #1

### 样例输入 #1

```
12 24
```

### 样例输出 #1

```
10
```

## 样例 #2

### 样例输入 #2

```
123 789
```

### 样例输出 #2

```
84
```

## 提示

对于 $40\%$ 的数据，$1\leq l\leq r\leq 10$。

对于 $70\%$ 的数据，$1\leq l\leq r\leq 1000$。

对于所有数据，$1\leq l\leq r\leq 10^6$。
